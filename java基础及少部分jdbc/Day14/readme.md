###1.异常
分类 ： 编译时期异常（已检查的异常），  运行时期异常  （未检出异常）
异常处理：2个办法  ，自己内部处理   ，扔出去 （给调用者）
try {可能发生异常 的代码}catch（异常类型 e){捕获处理} catch（）{}      finally{ 释放资源}、
注意：大异常放小异常的后面
运行时期的异常 不需要throws
编译时期的异常需要 throws
自己抛出异常： throw new 异常 （e）；
throw: 手动抛出一个异常，方法里              ；    throws ：方法声明上   告诉父类我的代码可能发生异常 ，
注意： 方法重写里的异常 ：子类异常可以是父类异常，也可以是父类异常的部分，也可以是父类异常的子类，也可以不写，
